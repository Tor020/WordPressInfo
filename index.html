    What is WordPress?

    WordPress began as a blogging platform but has since grown into a Content Management System Framework

    Features

  • Easy-To-Use Admin Panel

    • The admin panel comes packaged with the WordPress installation and makes it easy to manage the website with minimal technical expertise

  • It's Open-source

    • Open-source means that the code that makes up the software is freely distributed. This means that the software itself is free, and there is a large amount of support for it since the userbase is so massive.

  • Core Functionality extended via Plugins

    • Plugins are essentially applications that were coded and built to be integrated into WordPress and entail features. A huge amount of these are free.

      A few examples include plugins for
    • Search Engine Optimization
    • Website Backups & Security
    • Automatic Anti-Spam
    • Analytic Reports
  • Scales Extremely well and is Versatile

    • If you wanted to branch out and create a blog to help drive more potential business to your website Wordpress has built in functionality that allows you to have both a blog setup and regular pages.

    If you wanted to sell some kind of merchandise, and allow customers to place orders on your website, there are free WordPress Themes and Plugins that enable shopping cart support and can enable

  • Hundreds of Free WordPress Themes Available

    • WordPress utilize template files which affect the layout and style of your website. There are a great deal of high quality free WordPress themes avaiable to choose from which makes choosing the look and feel of your website easy.

    Bottom Line

    A WordPress website can be shaped to accomodate anything you want it to do, while still being simple enough to navigate and manage the upkeep of regardless of technical expertise

    Side Note

    WordPress in itself is secure, but it's very important to keep the website plugins up to date. Check the admin dashboard every 2 weeks at minimum to check for plugin and theme updates.
